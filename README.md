# EduTrack - Student Management System

EduTrack is built with Angular 16. It features a bold aesthetic using heavy black borders, vibrant lime accents, and high-performance data visualization.

---

## Key Features

* **Dynamic Dashboard:** Real-time student count and department distribution using Highcharts.
* **Student Management:** Full CRUD-capable list powered by AG Grid with custom lime-hover styling.
* **Smart Registration:** Reactive forms with 10-digit mobile validation and localStorage persistence.
* **Hybrid Data Sync:** Merges static JSON data with dynamic browser storage for real-time updates.

---

## Tech Stack

* **Framework:** Angular 16
* **Data Grid:** AG Grid Community
* **Charts:** Highcharts
* **Styling:** SCSS 
* **Icons:** BoxIcons
* **Notifications:** Hot-Toast
